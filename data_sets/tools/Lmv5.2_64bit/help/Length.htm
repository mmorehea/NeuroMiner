Length

This function returns the length of compartments by computing the distance between the two end points of a compartment.

Function Output Type : Numeric

Calculated : At Each Tracing Point

Command Line Usage Function : "-f11, 0, 0, 10.0"

Output:
Metric Total_Sum #Compartments
(considered)		 #Compartments
(discarded)		 Minimum Average Maximum S.D.
Length 16734 1548 (6) 1.11802 10.8103 656.878 22.0356
*All units are in microns

Output Interpretation :

The above example, shows the total length as 16734 microns.

Total_Sum = 16734 gives Total_Sum, Minimum = 1.11802 gives minimum, Maximum = 656.878 gives maximum for given input neuron.
