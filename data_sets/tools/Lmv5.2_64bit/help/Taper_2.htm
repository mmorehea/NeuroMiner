Taper_2

This function returns the Hillman taper. This is measured between two bifurcation points.
Formula : D a - D c
D a




Function Output Type : Numeric

Calculated : At Every Branch

Command Line Usage Function : "-f21, 0, 0, 10.0"

Output:
Metric Total_Sum #Compartments
(considered)		 #Compartments
(discarded)		 Minimum Average Maximum S.D.
Taper_2 10.7164 85 (1017) -0.333 0.126 0.8636 0.2235
*All units are ratios

Output Interpretation :

Hillman taper: given a branch it computes ratio between the final diameter (Da) and the initial one       (Dc) = (Da - Dc)/ Da.

Total_Sum = 10.7164 gives Total_Sum, Minimum = -0.333 gives minimum, Maximum = 0.8636 gives maximum for given input neuron.


References :
DE Hillman, Neuronal shape parameters and substructures as a basis of neuronal form, in: F Schmitt, ed., the neurosciences, 4th study program (MIT Press, Cambridge, 1979) 477-498
