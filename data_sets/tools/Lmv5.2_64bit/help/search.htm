LMSearch Panel

LMSearch Panel allows you to search for neurons on the basis of specific morphological functions.
On selecting LMSearch Panel, you will see the following screen.



The steps to follow to use LMsearch are as follows.
1) Select the function from the list of 44 functions on the left panel and click "Add" button. You may also add the function simply by double clicking it. Once the metric is selected it is added to the TABLE WINDOW on the right showing the selected function name in the second column, the other four columns consist of three drop down list boxes and one textbox.
2) Select the specific metric (Total_Sum, Minimum, Average, Maximum and SD) from the Metric column.
3) Select the specific operator (<, >, <=, >=, !=, ==) from the Opertor column.
4) Enter the number against which you wish to search the selected statistical function.
3) Choose the file or groups of files in which search is to be carried out by clicking "Choose File" button and then click "Start Search". The Lowest Panel will show the results of the search.
