PathDistance

This function returns the PathDistance of a compartment.

Function Output Type : Numeric

Calculated : At Each Tracing Point

Command Line Usage Function : "-f16, 0, 0, 10.0"

Output:
Metric Total_Sum #Compartments
(considered)		 #Compartments
(discarded)		 Minimum Average Maximum S.D.
PathDistance 399247 1554 (0) 0 256.915 1137.5 179.81
*All units are in microns

Output Interpretation :

PathDistance is which is the sum of lengths of each compartment from the soma point. The above example, shows the total PathDistance is 399247 microns.

Total_Sum = 399247 gives Total_Sum, Minimum = 0 gives minimum, Maximum = 1137.5 gives maximum for given input neuron.
