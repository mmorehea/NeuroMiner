Surface

This function returns surface of the compartment.

Formula: 2 x PI x r x h
Area of Cylinder


Function Output Type : Numeric

Calculated : At Each Tracing Point

Command Line Usage Function : "-f12, 0, 0, 10.0"

Output:
Metric Total_Sum #Compartments
(considered)		 #Compartments
(discarded)		 Minimum Average Maximum S.D.
Surface 17576 1548 (1) 0 11.3175 619.093 21.3962
*All units are in microns

Output Interpretation :

The surface area is computed using 2*pi*r*h, where r is the radius of compartment and h is the height. The above example, shows the total surface area is 17576 microns.

Total_Sum = 17576 gives Total_Sum, Minimum = 0 gives minimum, Maximum = 619.093 gives maximum for given input neuron.
