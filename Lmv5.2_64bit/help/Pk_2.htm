Pk_2

This function returns the same value as Pk, but with Rall_Power sets to 2 .
Formula : D n a + D n b
D n p

Function Output Type : Numeric

Calculated : At Bifurcation

Command Line Usage Function : "-f31, 0, 0, 10.0"

Output:
Metric Total_Sum #Compartments
(considered)		 #Compartments
(discarded)		 Minimum Average Maximum S.D.
Pk_2 154 96 (1458) 0.18 1.64727 2 0.44015
*All units are ratio

Output Interpretation :

The the Total_Sum of the Pk_2 doesn't make sense. Only the compartments leading to bifurcation are considered.

Total_Sum = 154 gives Total_Sum, Minimum = 0.18 gives minimum, Maximum = 2 gives maximum for given input neuron.
